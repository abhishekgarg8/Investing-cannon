# The Investing Canon
An agent-built repository of learnings from the 100 greatest public-markets investors, living and dead.
- One research task is executed per hourly run. Queue: `_admin/TODO.md`. History: `_admin/LOG.md`. Standards: `_admin/RESEARCH_SPEC.md`. Procedure: `_admin/HOURLY_RUNBOOK.md`.
- Each investor gets a folder under `investors/` with 8 standard documents (see spec).
- Cross-investor insights accumulate in `synthesis/`. Master directory: `INDEX.md`.
